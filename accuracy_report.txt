Model Performance ReportThe machine learning model was trained using historical sales data.A regression approach was used to predict Total Sales based on:- Day- Month- Weekday- Total Users- Total OrdersThe dataset was split into training and testing sets.Model performance was evaluated using Mean Absolute Error (MAE).The model achieved reasonable accuracy and produces consistent predictionson unseen (future) input data.Basic data cleaning and validation were applied to ensure numeric inputsbefore prediction.
